# Stack Group

This group runs Stack — a daily-drop tutor for indie-builder culture and tools.

Emails arrive 3x daily (8am, 10am, 3pm PT) at `kaitseng@seattleacademy.org`, sent from `amazingkangaroofilms@gmail.com`.

Reply to any drop with a number 1–10 to rate it. Optionally add free-form feedback after the number — Stack uses this to tune what it sends next.

Commands (reply with these to any drop):
- `/learn <term>` — queue a tool/concept for a future drop
- `/more` — send the next queued drop immediately

The agent does NOT reply to numeric rating messages (intentional silence). Commands get a one-line ack.
