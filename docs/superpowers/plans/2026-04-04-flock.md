# Flock Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a single self-contained HTML demo of Flock — a family travel itinerary builder that matches activities to family members — for a business planning competition, displayed on a projector.

**Architecture:** One `flock.html` file containing inline CSS (variables, animations, layout) and inline JS (screen navigation, tab switching, demo data). Three screen `<div>`s toggled via JS. No external dependencies.

**Tech Stack:** HTML, CSS (keyframe animations, grid, flexbox, custom properties), vanilla JavaScript.

**Spec:** `docs/superpowers/specs/2026-04-04-flock-design.md`

---

## File Structure

Single file:
- **Create:** `flock.html` — the entire app (HTML structure, `<style>` block, `<script>` block)

No other files. No build step. Open in browser to run.

// === Render Screen 1: Family Setup ===
function renderFamilyCards() {
  const container = document.getElementById('family-cards');
  container.innerHTML = '';

  FAMILY.forEach(member => {
    const card = document.createElement('div');
    card.className = 'member-card';
    card.style.borderTopColor = member.color;
    card.innerHTML = `
      <div class="member-card-header">
        <div class="member-card-identity">
          <div class="avatar lg" style="background:${member.gradient}">${member.name[0]}</div>
          <div>
            <div class="member-card-name">${member.name}</div>
            <div class="member-card-role">${member.role} · ${member.age}</div>
          </div>
        </div>
        <button class="member-card-edit">✏️</button>
      </div>
      <div class="member-card-tags">
        ${member.interests.map(i =>
          `<span class="tag" style="background:${member.colorLight};color:${member.colorText}">${i.emoji} ${i.label}</span>`
        ).join('')}
      </div>
      <div class="member-card-meta">
        <span>⚡ ${member.energy} energy</span>
        <span>♿ ${member.accessibility}</span>
      </div>
    `;
    container.appendChild(card);
  });

  // Add member placeholder card
  const addCard = document.createElement('div');
  addCard.className = 'add-member-card';
  addCard.innerHTML = `<div class="add-member-card-inner"><div class="plus">+</div><p>Add family member</p></div>`;
  container.appendChild(addCard);
}
